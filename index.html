10 Comadnmets of Linux


Comadment 1: PWD

pwd stands for Print Work directory. It shows the directory you’re currently in.

Comadment 2: ls

ls presents to you the contents of a particular directory – both files and directories.

Comadment 3: cd

Short for Change Directory, the cd command is behind your movement from one directory to another.

Comadment 4: mkdir

mkdirlets you create folders in anywhere you like in your Linux system – given you have got the necessary permission of course.

Comadment 5: rmdir

rmdir command allows you to delete specific folders from your system without any hassles.

Comadment 6: lsblk

lsblk terminal command will present you with a tree structure of your block devices.

Comadment 7: mount

mount command is used to mount the filesystem found on a device to big tree structure(Linux filesystem) rooted at ‘/‘.

Comadment 8: df

df command displays essential information about the disk space on your filesystem.

Comadment 9: uname

The uname command is used to obtain system information like name, version and other system specific details.

Comadment 10: ps

The ps terminal comand allows you to visualize what processes are currently run by your machine.
